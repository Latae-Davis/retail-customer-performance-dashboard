# Retail Customer Performance Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing retail sales and customer performance.

The dashboard provides insights into revenue, order volume, customer behavior, and sales trends across regions and product categories.

## Key Metrics
- Total Revenue
- Total Orders
- Average Order Value (AOV)

## Features
- Interactive filters (Region, Category, Segment, Date)
- Monthly sales trend analysis
- Sales breakdown by category
- Geographic performance by region

## Insights
- Revenue is concentrated in specific product categories
- Certain regions contribute disproportionately to total sales
- Sales trends show fluctuations that can indicate seasonality
- Customer segments impact overall purchasing behavior

## Tools Used
- Tableau Public (Web Authoring)
- Data Visualization & Dashboard Design
