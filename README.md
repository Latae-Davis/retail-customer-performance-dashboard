# Retail-KPI-Performance-Analysis

## Overview
This project analyzes retail performance data to evaluate sales trends, customer activity, and key business metrics. The goal was to use Excel to turn raw retail data into a clear performance analysis that highlights opportunities to improve sales, staffing decisions, and overall store performance.

## Business Problem
Retail leaders need visibility into performance in order to make better operational and sales decisions. This project was designed to answer questions such as:
- How are sales trending over time?
- Which products or categories are performing best?
- Which locations or time periods are underperforming?
- Are there patterns in customer transactions that could help improve performance?
- What KPIs should leadership monitor regularly?

## Objectives
- Clean and organize retail performance data
- Track key sales and customer KPIs
- Identify high-performing and low-performing areas
- Summarize trends in a business-friendly format
- Provide recommendations to improve retail performance

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- Formulas
- Tableau

## Dataset
The dataset used for this project includes sample retail performance data with fields such as:
- transaction date
- store location
- product category
- units sold
- revenue
- average transaction value
- customer count
- sales representative

Note: This project uses sample data created for portfolio purposes.

## Process
1. Cleaned and reviewed the raw retail data
2. Standardized fields for reporting and analysis
3. Built KPI calculations for sales, transactions, and customer trends
4. Used pivot tables and charts to identify patterns by category, location, and time period
5. Summarized findings and created recommendations for business stakeholders

## Key Metrics
- Total Revenue
- Units Sold
- Average Transaction Value
- Customer Count
- Sales by Category
- Sales by Location
- Monthly Sales Trend

## Key Findings
- Revenue performance varied across time periods, with some months showing stronger sales activity than others.
- Certain product categories drove a disproportionate share of total revenue.
- Some store locations underperformed compared to others, suggesting opportunities for targeted support.
- Average transaction value provided useful insight into customer purchasing behavior.
- KPI reporting made it easier to identify where operational or sales improvements were needed.

## Recommendations
- Monitor category performance regularly to identify top revenue drivers
- Review underperforming locations for staffing, merchandising, or promotional opportunities
- Track average transaction value alongside total revenue
- Use KPI reporting to support planning, scheduling, and sales strategy decisions
